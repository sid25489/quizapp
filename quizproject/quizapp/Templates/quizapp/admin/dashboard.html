📥 Upload Options & Format Guidelines

Choose how you want to add questions to your quizzes:

Manual Entry

Create individual quizzes with manually entered questions. Best for small, curated question sets.

CSV Format

Upload CSV files with columns: question, option1, option2, option3, option4, correct_option

JSON Format

Upload JSON files with quiz objects containing questions and choices arrays.

Excel Format

Upload .xlsx files with structured data. Each row = 1 question with options.
